# gulp-requirejs
包含gulp、requirejs、browserSync、arttemplate、sass、jquery、mockjs、babel的前端框架
## usage
<pre><code>
/* 安装依赖 */
npm i

/* 开发模式 */
gulp dev

/* 项目打包 */
gulp package
</code></pre>
